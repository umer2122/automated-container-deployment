# Automated Container Deployment and Administration in AWS

## Project Overview
This project demonstrates automated cloud infrastructure provisioning and container deployment using:

- Terraform (Infrastructure as Code)
- Ansible (Configuration Management)
- Docker (Containerization)
- GitHub Actions (CI/CD)
- AWS EC2 (Cloud Infrastructure)

## Architecture
Developer → GitHub → GitHub Actions → Terraform → AWS EC2 → Ansible → Docker Container → Web Application

## Features
- Automated EC2 provisioning
- Automated Docker installation
- Containerized Flask application
- CI/CD pipeline integration
- Security hardening practices

## Technologies Used
- AWS
- Terraform
- Ansible
- Docker
- GitHub Actions
- Python (Flask)
